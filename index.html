Esto es la imagen de un perrito triste por el pequeño retraso en la entrega de la tarea de una página web.

Que es un índice?

Es el archivo principal de una página web.

Para que sirve?

Sirve como punto de entrada para los visitantes además de permitir enlazar con otros archivos como CSS, JavaScript o páginas internas.
